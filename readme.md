## About

UABEANextVIEWTEST is a modified version of the original [UABEA](https://github.com/nesrak1/UABEA) by nesrak1. This fork adds Scene View functionality and a better UI.

- **[Nesrak1](https://github.com/nesrak1)** - Original UABEA/UABEANext creator
- **[Claude](https://claude.ai)** - Most implementation of code
- **[Xera](https://6b6t.com/bio)** - Other stuff
